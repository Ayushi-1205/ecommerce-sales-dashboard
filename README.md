# E-commerce Sales Storytelling Dashboard

An interactive data storytelling dashboard built with Python, Pandas, and Plotly Dash.

## Features
- KPI Cards (Revenue, Orders, AOV, Growth)
- Interactive filters by Region and Category
- Narrative section explaining key trends
- Time-series, bar, and funnel charts

## Tech Stack
- Python 3.x | Pandas | Plotly | Dash

## Setup
pip install -r requirements.txt
python app.py
